# 四路感應器

.smaller[

| 感應器 (左 ⮕ 右) | 十進制 |  二進制  |   動作   |
| :--------------: | :----: | :------: | :------: |
|   白 白 白 黑    |  `14`  | `0b1110` | 原地右轉 |
|   白 白 黑 黑    |  `12`  | `0b1100` | 原地右轉 |
|   白 黑 黑 黑    |  `8`   | `0b1000` | 原地右轉 |
|   黑 白 白 白    |  `7`   | `0b0111` | 原地左轉 |
|   黑 黑 白 白    |  `3`   | `0b0011` | 原地左轉 |
|   黑 黑 黑 白    |  `1`   | `0b0001` | 原地左轉 |
|   白 白 黑 白    |  `13`  | `0b1101` | 向右微調 |
|   白 黑 白 白    |  `11`  | `0b1011` | 向左微調 |

]
